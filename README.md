# Pneumonia Chest X-Ray Classification using Deep Learning

This repository contains Python code for automatic pneumonia detection 
from chest X-ray images using deep learning techniques.

Models implemented:
- Custom CNN
- CNN with Batch Normalization and Dropout
- Pre-trained ResNet50 (Transfer Learning)

Dataset:
Pneumonia Chest X-Ray Dataset (Kaggle)
